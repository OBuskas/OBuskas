# Hi, I'm Lucas Barbosa

Brazilian Front-end Developer based in Buenos Aires, Argentina.

I'm 28 years old and hold a degree in Systems Analysis and Development from FIAP. I currently work at Transfero Group on Dinx products, where I develop the BackOffice and subscription website for a financial education platform focused on children and families.

My main stack includes React, Next.js, TypeScript, Astro, Tailwind CSS, Shadcn/UI, and Ant Design, with a strong focus on scalable interfaces, UI quality, performance, and maintainable code.
